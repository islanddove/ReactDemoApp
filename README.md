# ReactDemoApp
React Demo App with an Express API Backend

# Prerequisites
Tested with nodejs and npm.

# Deployment
Clone this repo
Run 'npm start' from 'ReactDemoApp/' Directory to start the server.
Run 'npm start' from 'frontend/' to start the react application. It should automatically open in a browser window, if not go to http://localhost:3000/

To run tests, run 'npm test' from 'frontend/', only do so if the server has been started.
